Actividad 1

1.1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

La etiqueta ul sirve para hacer listas desordenadas adentro del texto, donde son pequeños detalles que no se muestra con mucha importancia.
Ej:

  • Azul
  • Blanco
  • Morado

1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías?

El ol tiene el mismo funcionamiento que la etiqueta ul, solo que se usa para las listas ordenadas. Que destacan mas en la pagina por dar pasos numericos.
Ej:

  1. Azul.
  2. Morado.
  3. Blanco.

1.3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías?

La etiqueta dl es para generar listas descriptiva, dandole un breve desarrollo a los materiales que se esta informando.
Ej:

Azul
Es un color primario, se le acompleja con el color del oceano.
Morado
Es un color derivado de la mezcla del color azul y rojo.
Blanco
Es un color que esta fuera del orden de colores, pero ayuda a que estos sean mas claros. Se le asemeja con el color de la puerza.

1.4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

La etiqueta li es que da la separacion dentro de los textos las listas.
Ej:

  • Azul
  • Morado
  • Blanco

    Actividad 2

    2.1- ¿qué es una url absoluta?

    Las URL absolutas proporcionan la URL completa del documento, incluido el protocolo, el nombre de dominio y la ruta como necesario. Debe usar una URL absoluta cuando apunta a un documento en la web.

    2.2- ¿qué es una url relativa?

    Las URL relativas describen la ruta a un archivo relativo al actual documento. Las URL relativas se pueden usar cuando se vincula a otro documento en su propio servidor.
